# FPGA Coursera Course
## Overview
This repository contains resources and code for the "FPGA Architecture Based Systems for Industrial Applications" course. The project leverages a Docker image for Vivado 2024 to facilitate a streamlined development environment.

## Docker Image
The Docker image for Vivado 2024 is available at:

Docker Hub - xilinx-vivado

You can pull the Docker image using the following command:

```

docker pull seu_usuario/xilinx-vivado:2024.1-x11

```
Look at my page in [docker hub](https://hub.docker.com/r/gusanagy/xilinx-vivado) for more instructions. 

OBS: The docker image needs to set the locale to en_US.UTF-8 and need to install gcc. Also you need a license to use the image. 

## Usage

* Pull the Docker Image: Use the command above to download the image.
* Run the Docker Container: Follow the instructions provided on the Docker Hub page for setting up and running the container.
* Access the Vivado Environment: Once inside the container, you can start using Vivado 2024 for your FPGA development tasks.
